Essa página está sendo estilizada com web fonts

O título da página foi estilizado com a web font Press Start 2P, oriunda do Google Fonts. O texto da página usa a fonte VT323.

Explicação sobre o código

O código dessa página está usando web fonts baixadas e salvas na pasta fonts desse repositório. Isso significa que essas fontes não estão sendo carregadas diretamente de uma url remota.
